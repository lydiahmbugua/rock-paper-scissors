# Rock Paper Scissors

A web-based implementation of the classic game, Rock Paper Scissors. This project focuses on implementing game logic, managing score state, and updating the UI dynamically based on player interaction.

## 🚀 Live Demo
[https://lydiahmbugua.github.io/rock-paper-scissors/]

## 📋 Features
- **Interactive Gameplay:** Users play directly against the computer.
- **Score Tracking:** Real-time updates for both player and computer scores.
- **Round Announcements:** Clear messaging indicating the winner of each round.
- **Reset Functionality:** Ability to restart the match once a winner is declared or at any time.

## 🛠 Technologies Used
- **Frontend:** HTML5, CSS3, Vanilla JavaScript (ES6+)
- **Logic:** Math.random() for computer selection, conditional control flow for winner determination.
